Tratamento de dados

parseInt: usado para fazer tranformação de dados, números inteiros.

parseFloat: Usado para fazer transformação de dados, porém decimais.

Number: Pode ser usado somente Number, o JavaScript se vira.

String(n): Uma String para Number.

n.toString(): Uma número em uma string.

Formatoss de String

s.length: Quantos caracteres a string tem.

s.toUpperCase: Tudo para maiusculo.

s.toLowerCase: Tudo para minusculo.

documeent.write(): Escrever na tela

s.toFixed: Para casa decimal

replace(".",","): para trocar um ponto por virgula
